Typical Morphology Problem Types

Type 1: Segment the Word

  • Task: Identify recurring pieces and their meanings.

  • Key strategy:

    • Align meanings first
    • Then align forms
    • Only segment when evidence is repeated

Type 2: Identify the Grammatical Function

  • Task: Decide what a recurring element does (plural? diminutive? tense?)
  • Key strategy:
    • Ask: What meaning is always present when this form appears?
    • Avoid: Guessing labels too early

Type 3: Predict New Forms

  • Task: Generate a word you have never seen.

  • Key strategy:

    • Apply the simplest rule that fits all known data
    • Check position and compatibility

Type 4: Resolve Apparent Irregularity

  • Task: Explain why forms do not look uniform.

  • Key strategy:

    • Ask: Whether the same grammatical meaning is expressed
    • by different phonological shapes
    • under different conditions
